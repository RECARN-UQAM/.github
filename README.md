# .github

This repository holds organization-wide defaults and community health files for the
[RECARN-UQAM](https://github.com/RECARN-UQAM) GitHub organization.

Files here are inherited by every repository in the organization that does not
provide its own version.

## Contents

| File | Purpose |
| --- | --- |
| [`profile/README.md`](profile/README.md) | Public profile shown on the organization landing page |
| [`contributing-guide.md`](contributing-guide.md) | How to contribute to RECARN-UQAM projects |
| [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md) | Expected standards of behavior for the community |
| [`AGENTS.md`](AGENTS.md) | Guidance for using AI assistants in contributions |
| [`PULL_REQUEST_TEMPLATE.md`](PULL_REQUEST_TEMPLATE.md) | Default template for pull requests |
| [`ISSUE_TEMPLATE/`](ISSUE_TEMPLATE) | Default templates for bug reports and feature requests |

## Notes

- Editing a file here updates the default across all organization repositories that
  do not override it.
- `profile/README.md` renders on the organization's public page and is separate from
  this file, which is only shown when viewing this repository directly.
