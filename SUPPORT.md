# Support

Thanks for your interest in RECARN-UQAM! This page explains where to go for help,
depending on what you need.

## I want to join or get involved

- Complete our [Membership Form](https://forms.office.com/r/BhrEg8yGGq).
- Follow the [GitHub Organization](https://github.com/RECARN-UQAM) to stay updated.
- Learn how to contribute in our [Contributing Guide](contributing-guide.md).

## I have a question about a project

- Check the project's `README` and documentation first.
- Search existing issues in the relevant repository — your question may already be
  answered.
- If Discussions are enabled on the repository, start a discussion there.

## I found a bug or want to request a feature

- Open an issue using the appropriate template:
  - [Bug report](.github/ISSUE_TEMPLATE/bug_report.md)
  - [Feature request](.github/ISSUE_TEMPLATE/feature_request.md)

## I need to report a security issue

Please do not open a public issue. Follow our [Security Policy](SECURITY.md) instead.
